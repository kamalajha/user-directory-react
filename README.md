# User Directory React App
A simple React application that displays a list of users with a search functionality. You can search users by name, and each card allows expanding to view additional information

## Features
- Search users by name
- Clean and responsive UI.
- Card-based layout for each user
- Show/Hide user details within the same card
Component-based architecture (SearchBar, UserCard, UsersPage)

## Tech Stack
- React 
- css
- Javascript

## How to Run
1. Clone the repository: ```bash git clone https://github.com/kamalajha/user-directory-react.git

2. Install Dependency

3. Start the app:
  npm start
